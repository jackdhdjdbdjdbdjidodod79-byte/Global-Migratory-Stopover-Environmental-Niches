Data Availability Statement

All environmental variables, processed stopover datasets, and derived analytical products used in this study are publicly available through this GitHub repository:

https://github.com/xxxxxxx/Global-Migratory-Stopover-Environmental-Niches

The repository includes:

• Processed stopover-site dataset
• Environmental predictor variables
• Statistical outputs
• Figure-generation scripts

Raw bird tracking data were obtained from publicly available migration databases and were processed following the workflow described in the manuscript.
